# Oracle-s-SCIF-MIL-STD-Portable-Data-Center
Oracle Roving Edge Station is a secure expeditionary data center enabling applications in the field, connected or disconnected. 

Meets MIL-STD-810H for Ruggedness and MIL-STD-461 EMC Compliance

SCIF Top Secret Rated: Biometric Entrance, 360 Video surveillance (30 Hr Recording), GSA X-10 Approved Lock, Tempest Rated up to 30 meters.

2x24KW Racks to run AI workloads to be paired with portable soldier carried KLAS Voyager Data downloads/uploads.
